#  项目名称：外卖平台每日经营情况分析

# 项目简介
本项目基于某外卖平台提供的用户订单数据，运用数据分析技术进行深入探索，完成数据清洗、可视化分析，并提出业务洞察和优化建议。项目使用 Excel/Tableau/SQL/Python 等工具，作为数据分析/量化研究方向的入门实战项目。

#项目结构
├── data/ # 数据文件
├── images/ # 可视化图表、Tableau 截图等
├── 美团饿了么上海某门店数据分析表/ # 结果文件
└── README.md # 项目说明文档
# 使用工具和技术
- Excel：基础函数（SUMIFS、IF、VLOOKUP、INDEX-MATCH）、数据透视表
- SQL：数据筛选、聚合、窗口函数
- Tableau：仪表盘设计、交互式图表
#分析目标与结论
- 用户下单行为的时段和频率分析
- 订单分布
- 商家/骑手的绩效分析
- 提出运营优化建议（如促销时间、用户留存等）
